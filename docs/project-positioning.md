# Project Positioning

Gongkao Mistake Review PWA is an early-stage, local-first learning tool for civil service exam and Xingce mistake review.

It exists because many learners do a large amount of practice, but their mistakes do not always become reusable review material. A normal mistake notebook may save the question and answer. This project tries to split each mistake into fields that can be reviewed later:

- what the mistake came from;
- which subject and module it belongs to;
- what the wrong path was;
- what the core trap was;
- what formula or method should be remembered;
- what summary should be reviewed next time;
- when it should be reviewed again.

The project is mistake-centered rather than question-bank-centered. It expects users to provide their own content through manual entry, pasted notes, draft import, or experimental image workflows.

## What It Is

- A personal study workflow tool.
- A local-first PWA.
- A structured mistake-card organizer.
- A review scheduler with simple feedback states.
- A place to keep user-owned learning data and export it as JSON.
- A cautious AI-assisted organizer where users confirm the final result.

## What It Is Not

- It is not an online question bank.
- It does not distribute official civil service exam questions.
- It does not provide official answers or exam authority material.
- It does not guarantee score improvement.
- It is not an autonomous AI tutor.
- It is not mature cloud storage.

## Core Narrative

Turn mistakes into reviewable learning assets.

The project should be presented as a serious exploration of local-first study workflows, not as a polished commercial product.
